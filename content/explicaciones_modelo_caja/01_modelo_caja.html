Modelo de Caja (Box Model)

Padding

Espacio interno entre el contenido y el borde.

Por partes:

Contenido
padding-top: 25px;
padding-right: 100px;
padding-bottom: 50px;
padding-left: 10px;

Abreviado (4 valores):

Contenido
padding: 25px 100px 50px 10px; /* top right bottom left */

Abreviado (3 valores):

Contenido
padding: 25px 100px 50px; /* top right/left bottom */

Abreviado (2 valores):

Contenido
padding: 25px 100px; /* top/bottom right/left */

Abreviado (1 valor):

Contenido
padding: 25px; /* todos los lados */

Border

Borde visible alrededor del padding y contenido.

Por partes:

Contenido
border-width: 5px;
border-style: solid;
border-color: blue;

Abreviado:

Contenido
border: 5px solid blue;

Por partes (estilos diferentes):

Contenido
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;

Por partes abreviado:

Contenido
border-top: 5px dotted red;
border-right: 5px solid blue;
border-bottom: 5px dotted red;
border-left: 5px solid blue;

Bordes redondeados:

Contenido
border-radius: 5px;

Margin

Espacio externo fuera del borde.

Por partes:

Contenido
margin-top: 25px;
margin-right: 100px;
margin-bottom: 50px;
margin-left: 10px;

Abreviado (4 valores):

Contenido
margin: 25px 100px 50px 10px; /* top right bottom left */

Abreviado (3 valores):

Contenido
margin: 25px 100px 50px; /* top right/left bottom */

Abreviado (2 valores):

Contenido
margin: 25px 100px; /* top/bottom right/left */

Abreviado (1 valor):

Contenido
margin: 25px; /* todos los lados */
