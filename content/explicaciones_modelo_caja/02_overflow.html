Propiedad Overflow

Controla el comportamiento de un contenedor cuando su contenido es más grande que el contenedor mismo y se desborda.

overflow: visible

Este es el valor por defecto. El contenido que se desborda simplemente se muestra fuera del contenedor, sobresaliendo visualmente.

Este es un texto muy largo que se desborda del contenedor porque el contenido es más grande que el área disponible.
overflow: visible;

overflow: hidden

El contenido que se desborda está oculto y no es visible. Esto puede ser útil cuando no quieres que el contenido que sobresale sea visible.

Este es un texto muy largo que se desborda del contenedor. Con overflow: hidden, la parte que no cabe se corta y no es visible. Solo verás el contenido que cabe dentro del área definida.
overflow: hidden;

overflow: scroll

Se agrega una barra de desplazamiento al contenedor, permitiendo al usuario desplazarse para ver el contenido que está fuera de la vista.

Este es un texto muy largo que se desborda del contenedor. Con overflow: scroll, siempre verás barras de desplazamiento tanto horizontal como vertical, incluso si el contenido cabe perfectamente. Puedes desplazarte para ver todo el contenido.
overflow: scroll;

overflow: auto

Similar a scroll, pero se agregarán barras de desplazamiento solo cuando sea necesario. Si el contenido se ajusta dentro del contenedor, no se mostrarán barras de desplazamiento.

Este es un texto muy largo que se desborda del contenedor. Con overflow: auto, las barras de desplazamiento solo aparecen cuando el contenido realmente se desborda.
overflow: auto;
