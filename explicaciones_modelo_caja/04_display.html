Propiedad Display

Determina cómo se muestra un elemento en el diseño de una página web. Puede tomar varios valores, cada uno con su propio comportamiento.

display: block

  • Generan un nuevo bloque de formato, comenzando en una nueva línea
  • Ocupan todo el ancho disponible si no se establece width
  • Se adaptan al alto del contenido si no se establece height
  • Ejemplos de bloque son: párrafos (<p>), encabezados (<h1>-<h6>), divisiones (<div>)

display: inline

  • No inician una nueva línea
  • Ocupan el ancho necesario para su contenido
  • No les afecta width o height
  • Ejemplos: enlaces (<a>), imágenes (<img>), etiquetas de formato <b> <em> <i> <sup> <span>

display: none

El elemento no se muestra en absoluto. Es como si el elemento no existiera en el diseño de la página. Esto puede ser útil para ocultar temporalmente elementos.
