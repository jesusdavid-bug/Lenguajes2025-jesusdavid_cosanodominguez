Propiedad Box-Sizing

Controla cómo se determinan sus dimensiones totales, incluyendo el ancho y la altura.

Comparación

Mismo elemento con diferentes valores de box-sizing:

width: 500px;
height: 100px;
padding: 10px;
border: 20px;

box-sizing: content-box

Contenido

500px + 20px (padding) + 40px (border) = 560px

box-sizing: content-box;

box-sizing: border-box

Contenido

500px (incluye padding y border)

box-sizing: border-box;

Explicación

content-box (por defecto):

El width y height solo incluyen el contenido. El padding y border se suman al tamaño total.

Tamaño total = 500px + 20px (padding) + 40px (border) = 560px

border-box:

El width y height incluyen el contenido, padding y border. El tamaño total es el especificado.

Tamaño total = 500px (ya incluye todo)
